# 📚 Zaptude – Aptitude Learning Platform

Zaptude is an aptitude learning platform designed to provide a structured and interactive learning experience for problem-solving practice.

---

## 🚀 Features
- Structured question system for organized learning  
- Clean and responsive user interface  
- Interactive design focused on user engagement  
- Mobile application under active development  

---

## 👨‍💻 My Contributions
- Converting the existing web application into a mobile application  
- Adapting UI/UX for mobile responsiveness and usability  
- Integrating backend services using Supabase  
- Optimizing performance for a smooth mobile experience  

---

## 🛠 Tech Stack
- Dart  
- Flutter (Android Studio)  
- Supabase
