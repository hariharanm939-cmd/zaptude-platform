# Zaptude Lite - Core Logic Demo

This folder contains only the essential source code for the Zaptude platform to demonstrate the system architecture.

### 📱 Frontend (Mobile)
- **`mobile_main.dart`**: The application's entry point and state management setup.
- **`mobile_login_ui.dart`**: The primary authentication UI and branding style.

### ⚙️ Backend (API)
- **`backend_server_core.js`**: The main Express server configuration and middleware.
- **`backend_auth_logic.js`**: Security logic for JWT-based user authentication.
- **`database_structure.sql`**: The PostgreSQL schema for students and placement data.
